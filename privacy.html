Политика конфиденциальности

Версия: 1.0 · Дата последнего обновления: 13 декабря 2025

Настоящая Политика конфиденциальности описывает, какие данные могут обрабатываться при использовании сервиса Heretic (далее — «Сервис»), и на каких основаниях. Мы исходим из принципа минимизации данных: собираем только то, без чего сервис не работает.

1. Какие данные мы не собираем

Сервис и приложение Heretic не собирают и не хранят:

  • содержимое пользовательского трафика;
  • историю посещений ресурсов/сайтов;
  • переписки, файлы, пароли;
  • рекламные идентификаторы и поведенческие профили;
  • ФИО, адрес проживания, паспортные данные.

2. Какие данные могут обрабатываться

Для работы Telegram-бота Сервис может обрабатывать технические сведения:

  • Telegram ID пользователя (идентификатор аккаунта в Telegram);
  • статус доступа (например, активен/истёк), срок действия и связанные с доступом параметры.

Эти данные используются только для выдачи и управления доступом (например, чтобы не выдавать пробный доступ повторно и корректно показывать активные токены).

3. Обращения в поддержку

Если пользователь обращается в поддержку по собственной инициативе, он может добровольно передать дополнительные сведения (например, описание проблемы). Такие сведения используются исключительно для ответа и решения обращения.

4. Серверная инфраструктура и аренда мощностей

Сервис использует арендованную серверную инфраструктуру у сторонних провайдеров. Мы не ведём целенаправленный сбор пользовательских логов и не храним содержимое трафика. В пределах, допустимых технической средой и настройками инфраструктуры, применяется принцип минимизации данных.

5. Передача данных третьим лицам

Мы не продаём и не передаём данные пользователей третьим лицам, за исключением случаев, прямо предусмотренных применимым законодательством.

6. Сроки хранения

Telegram ID и связанные с доступом технические данные хранятся ровно столько, сколько необходимо для предоставления и администрирования доступа, либо до отзыва/истечения доступа, если иное не требуется по закону.

7. Безопасность

Мы применяем разумные организационные и технические меры для защиты информации, включая ограничение доступа, разделение прав и контроль изменений.

8. Изменения политики

Мы можем обновлять Политику. Актуальная версия всегда опубликована по текущему адресу страницы. Продолжение использования Сервиса означает согласие с обновлённой редакцией.

9. Контакты

По вопросам конфиденциальности вы можете обратиться в поддержку через Telegram или иным способом, указанным в интерфейсе Сервиса.

Примечание: данный документ носит информационный характер и описывает фактический подход Сервиса к данным.
